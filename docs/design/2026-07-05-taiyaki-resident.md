# たい焼き住民 設計ドキュメント

- 日付: 2026-07-05
- 状態: オーナー承認済み
- 目的: 3体目の住民「たい焼き」を追加する

## コンセプト: 量産式の初適用

「形状テンプレート × フレーバーパレット = 1住民」の量産式を初めて適用する。
ラムネ魚の形状（ドット配置）をそのまま使い、たい焼きフレーバー（こんがり焼き菓子）のパレットを適用する。

## スプライト

- 形状: ラムネ魚 frame1 のグリッドを流用する。ドット配置は不変
- パレット（こんがり基調）:

| 記号 | 色 | 意味 |
|---|---|---|
| H | #F6D394 | ハイライト |
| L | #E9B665 | 明部 |
| S | #D9A254 | 基本色 |
| M | #7A4238 | あんこ（腹の焼き目位置に中身が覗く） |
| E | #8A5C2B | 縁・焼き色 |
| O | #3A2A1E | 目 |

## アニメーション

- 剛体（焼き菓子は曲がらない）。1フレームのみ・`frameIntervalMs: 0`
- 動きは泳ぎの上下ボブのみで表現する

## 泳ぎパラメータ

- speed: 0.18
- bobAmplitude: 3
- bobFrequency: 0.0012

## 誕生順

- BIRTH_TABLE の3番目に追加（ramuneFish → strawberryJelly → taiyaki のループ）

## スコープ外

- 常駐化（プロップとセット・滞在スケジューラの回で扱う）
- 図鑑
