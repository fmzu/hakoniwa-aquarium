# hakoniwa-aquarium

ブラウザで遊ぶドット絵の箱庭ゲーム。不思議な生物が棲む「海域」で、主人公の子ネッシーがリング状の海を泳ぎ、影の魚を食べると新しい住民が生まれて海域がにぎやかになっていく。

## 遊び方

- **線描き**: 画面をなぞると道のりが描かれ、子ネッシーがそれに沿って泳ぐ
- **タップ**: 1回タップでその場所へ向かう（線描きの簡易版）
- **捕食**: 影の魚（シルエットの餌）に頭から近づくと食べられる。胴では食べられない
- **誕生**: 影の魚を5匹食べると、新しい住民が1体生まれて海域を泳ぎはじめる

## 公開URL

## 開発

```sh
bun install      # 依存インストール
bun run dev      # 開発サーバー起動
bun test         # テスト実行
bun run lint     # lint（Biome）
bun run build    # ビルド
```
